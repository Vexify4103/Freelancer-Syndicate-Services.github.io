## Our Structure

FSS operates with a simple rank structure designed to be flat and welcoming while still recognizing experienced contributors.

## Ranks

| Rank | How to Earn | Notes |
|------|-------------|-------|
| **Recruit** | Join the org | Starting rank for all new members |
| **Member** | Active for 2+ weeks | Participate in org ops or Discord |
| **Senior Member** | Active for 2+ months | Contribute to org activities regularly |
| **Officer** | Appointed by leadership | Trusted members who help run operations |
| **Founder** | Founding members only | Organization leadership |

## How to Advance

There's no formal application process for advancing beyond Recruit — we promote based on what we see. Show up, participate, help others, and you'll move up naturally.

If you think you've been overlooked, just ping an Officer in Discord.

## Roles and Divisions

Beyond ranks, FSS members often specialize in a division:

- **Trade Division** — Cargo hauling, commodity trading, smuggling
- **Mining Division** — Ore and surface mining operations
- **Combat Division** — Bounty hunting, escort, PvP, piracy defense
- **Exploration Division** — Scanning, mapping, jump point scouting

You can be part of multiple divisions. Divisions organize their own operations and have dedicated channels in Discord.

## Code of Conduct

We keep this simple:

1. **Respect everyone** — members, recruits, and outsiders alike
2. **Don't grief** friendly players unprovoked
3. **Communicate** — if you're going to be inactive, let us know
4. **Have fun** — this is a game

Breaking the code of conduct will result in a warning, then removal from the org at officer discretion.
